## Cara Install

```shell
┌─[nightsec@greyxploiter]─[/var/www/html/sekolah]
└──╼ $ git clone https://github.com/greyxploiter/sekolah.git
┌─[nightsec@greyxploiter]─[/var/www/html/sekolah]
└──╼ $ php init
┌─[nightsec@greyxploiter]─[/var/www/html/sekolah]
└──╼ $ composer update
```

Kemudian Upload [Database](fix_db.sql) ke server.
