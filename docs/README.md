# Sekolah Apps

## Features
* Menggunakan 3 role
	- Admin
	- Guru
	- User
* Jadwal
* Materi
* Materi Soal
* Materi Jawaban
* Kelas
* Artikel Berita
* Galeri
* Dan Lainnya.

## Cara Install
```
┌─[nightsec@greyxploiter]─[/var/www/html/sekolah]
└──╼ $ git clone https://github.com/greyxploiter/sekolah.git
┌─[nightsec@greyxploiter]─[/var/www/html/sekolah]
└──╼ $ php init
┌─[nightsec@greyxploiter]─[/var/www/html/sekolah]
└──╼ $ composer update

And Then Upload file fix_db.sql to your server.
```
